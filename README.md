# BoardEase Residences

BoardEase Residences is a premium property management system designed to provide a seamless experience for both landlords and tenants. Manage payments, rooms, and maintenance requests with ease through a high-fidelity, mobile-first interface.

## Features

- **For Landlords**: Manage room inventory, track rent payments, broadcast announcements, and handle maintenance tickets through a unified dashboard.
- **For Tenants**: Pay rent, track history, view official receipts, and report maintenance issues instantly.

## Tech Stack

- **Frontend**: React, Tailwind CSS, shadcn/ui.
- **Backend/Database**: Supabase.
- **Icons**: Lucide React.

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`.
3.  Set up environment variables in `.env`.
4.  Run the development server: `npm run dev`.
